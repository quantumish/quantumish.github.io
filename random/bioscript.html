Biology FlyTracker Script

Download link here.

This is a simple Python script for getting rows of the FlyTracker data that are within a specific range (a rectangular region defined by the bottom left corner and top right corner).

To use this script, you first need to export your data as a CSV. Then, run the python script in the terminal with the following command.

python2 bioscript.py FILE_PATH X_ROW Y_ROW BOTTOM_X BOTTOM_Y TOP_X TOP_Y OUTPUT_PATH

You'll need to replace some things:

  • FILE_PATH with the path to your CSV file (dragging it into the terminal can autofill this!)
  • X_ROW with whichever row has the fly's x coordinate in your data (remember to count up from 0, not 1)
    • Same for Y_ROW
  • BOTTOM_X, BOTTOM_Y should be replaced with the x and y points for your bottom corner of the square, same but for the top corner with TOP_X, TOP_Y.
  • OUTPUT_PATH with a path to a file you want the program to produce.

You can then import the CSV file the script produces to analyze it with another piece of software like Excel or Google Sheets - or you could just stick with Python for more flexibility.

Finally, here's the source code for anyone interested:

import csv
import sys

def get_range(path, xrow, yrow, bound_lower, bound_upper, outpath):
    f = open(outpath, "w")
    with open(path, newline='') as csvfile:
        reader = csv.reader(csvfile, delimiter=' ', quotechar='|')
        for num, row in enumerate(reader):
            if num == 0: continue
            contents = row[0].split(",")
            if contents[xrow] == '' or contents[yrow] == '': continue
            coords = (float(contents[xrow]), float(contents[yrow]))
            if coords > bound_lower and coords < bound_upper:
                f.write(', '.join(row) + '\n')
    f.close()
